Basic concepts and jargon regarding the internet

Browser - This is a program that accesses and displays webpages. The browser requests the relevant data based on user input (such as a web addresse) from a server.

Server - A computer that hosts files. A server will idle until it recieves a request, sending (hopefully) the appropriate file (a sort of webpage).

Webpage - This is a document (computer file) written in a language called HyperText Markup Language (HTML) including CSS and others. The user's browser parses this code and displays the page accordingly.

HTTP (HyperText Transfer Protocol) - The conventional method governing the transfer of information in the form of webpages over the internet.

A little about HTML..

The structure: (without the space between the tags).

< !DOCTYPE HTML > - Instructs user's browser on version of html being used.
< html > - Surrounds entire document..
< head > - Includes metadata, javascript and css and scripts.
< title > Title of the document. < /title >
< /head >
< body > - Content of actual document.
< /body >
< /html >

A few more basic tags including the use of attributes:

< b > < /b > - Bold text.
< em > < /em > - Emphasis (italicize).
< a > < /a > - Anchor. Creates hyperlinks.
Example; < a href="address of website" > text that is hyperlinked. < /a > href is the attribute of the anchor tag.
< img > - Image (no closing tag).
Example; < img src="URL (of image)" alt="text to display if image cannot be shown" > both src and alt are attributes.
< br > - Line break (no closing tag).
< p > < /p > - Paragraph.
< div > < /div > - Container for elements.
< span > < /span > - container for text. Blank.

Last note: Block elements are displayed with line breaks at the beginning and end. Inline elements are not.
